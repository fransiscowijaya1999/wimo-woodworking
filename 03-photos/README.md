# 03 — Photos

Reference and progress photos.

- Name descriptively with a date prefix: `2026-06-18-drawer-frame-glue-up.jpg`.
- Photos can bloat a git repo. Before committing, compress to under ~500KB where
  possible, or use Git LFS / external storage — see the image-handling notes in the root
  `README.md`. The `.gitignore` already has commented-out rules to exclude image types if
  they get large.

Claude can read and describe photos placed here, but cannot take or draw them.
