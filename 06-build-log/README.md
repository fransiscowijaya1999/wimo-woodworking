# 06 — Build Log

A dated journal of construction progress. One file per entry keeps diffs clean and history
readable.

- Copy `TEMPLATE.md` to a new file named `YYYY-MM-DD-short-slug.md`
  (e.g. `2026-06-18-drawer-carpentry-start.md`).
- Commit build-log entries with the `Build:` prefix.
- When a build step reveals that a design assumption was wrong, also update
  `../01-claude-project/DESIGN_SPECIFICATIONS.md` (and `REFERENCE_DATA.md` if a constant
  or error pattern changed) so the spec never drifts from reality.
