# 04 — Vendors

Supplier and purchase information: where materials and hardware come from, prices, and
order history.

- Keep entries **generic enough for a public repo** — region-level, not specific brand
  portfolios or competitive detail (see the Privacy Notice in the root `README.md`).
- For receipts or records containing personal info, name the file `private-*` (git-ignored)
  or uncomment the receipts rule in `.gitignore`.

Suggested per-vendor note: material/hardware supplied, typical price, unit, lead time,
and any quality observations.
